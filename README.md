## Instalação

Todo o processo, a partir do item 2, será feito em linha-de-comando. Use o terminal de sua escolha no seu sistema operacional.

1. Baixe a gema compilada da [página de _releases_](https://github.com/Nereare/iamspe/releases/latest) do projeto;
2. Vá até o diretório da gema baixada;
3. Instale a gema;
4. Use a gema! &#x1f389;

```bash
cd /path/to/gem/file
gem install iamspe-x.y.z.gem
```

## Uso

Esta gema vem com dois executáveis incluídos no _PATH_ quando instalada: `chefia` e `porta`.

O uso implica apenas digitar o nome do executável em questão no seu _bash_, _cmd_, _PowerShell_, ou linha-de-comando de escolha.

## Contribuindo

1. Crie um _fork_;
2. Crie um _branch_ para a funcionalidade a implementar: `git checkout -b nova-funcionalidade`;
3. Cometa as mudanças: `git commit -am 'Mudanças criadas'`;
4. Submeta as mudanças cometidas: `git push origin nova-funcionalidade`;
5. Abra um _pull request_! &#x1f64f;
